# 페이지 이미지 드롭 폴더

원본 구글 사이트의 일부 스크린샷/영상 캡처는 학교 구글 계정 인증이 걸려 있어
자동 이관이 되지 않습니다. 운영자가 원본에서 직접 내려받아 아래 폴더에 넣으면
해당 페이지에 자동으로 표시됩니다(`PageImages` 컴포넌트).

## 넣는 방법
- 파일명을 `01.png`, `02.png` … 순서로 저장 (png/jpg/webp 지원)
- 해당 폴더에 넣고 커밋 → 배포되면 그 자리에 이미지가 나타남
- 파일이 없으면 그 자리에는 "이미지 자리" 안내 박스가 보임

## 폴더 ↔ 위치
| 폴더 | 페이지 위치 |
|---|---|
| `manual/1-neis/`     | 사전 준비사항 안내 > 1. NEIS 자료 연동 |
| `manual/2-signup/`   | 사전 준비사항 안내 > 2. 교육디지털원패스 회원가입(가입 방법 영상·이미지) |
| `eduonepass/signup/` | 교육디지털원패스 > 회원 가입 방법 |
| `study/`             | 스스로 학습 안내 (이미 이관됨: poster.png, portal-button.png) |
