# 원본 ↔ 신규 사이트 시각적 대조 자가 체크리스트

이 문서는 운영자가 **기존 원본 사이트(구글 사이트)**와 **신규 구축 사이트(Cloudflare Pages)**를 눈으로 직접 보며 1:1 비교 검수할 때 활용하는 상세 시각적/기능적 체크리스트입니다.

---

## 0. 검수 준비

*   [ ] 브라우저 창 2개를 좌우로 분할하여 배치합니다.
    *   **왼쪽 화면 (원본)**: `https://ai-dt.net` (도메인 전환 전 기준)
    *   **오른쪽 화면 (신규)**: `https://ai-dt-net.pages.dev/`
*   [ ] 브라우저 개발자 도구(F12)를 열어 **모바일 뷰 에뮬레이터**를 활성화하고 모바일 레이아웃(너비 375px~420px)도 수시로 교차 점검합니다.
